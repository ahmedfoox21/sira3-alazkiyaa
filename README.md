# ⚔️ صراع الأذكياء - لعبة المسابقات الجماعية أونلاين

صراع الأذكياء هي لعبة متكاملة للمنافسة والمسابقات الجماعية المباشرة (Real-time)، تم بناؤها بالكامل للهواتف الذكية مع دعم كامل للاتجاهات العربية (RTL) وتصميم حديث يدمج ألوان النيون الذهبية والبنفسجية.

## 🚀 المميزات والتقنيات المستخدمة

- **واجهة المستخدم (Frontend)**: 
  - React 19 مع TypeScript.
  - Tailwind CSS للتنسيق السريع والأداء السلس على الهواتف.
  - `motion` لتفاعلات شاشة البدء والانتقالات السلسة.
- **الخلفية وقناة الاتصال (Backend & Sockets)**:
  - Node.js & Express.
  - Socket.IO للمزامنة اللحظية لعداد الثواني ونقاط الإجابات وحالة الغرف النشطة.
- **الصوتيات والاتصال الحي**:
  - Web Audio API لتركيب تأثيرات الصوت الصاخبة للبزرر، وتكتكة الوقت، والنجاح محلياً دون بطء تحميل أصول الصوت.
  - WebRTC signaling لمحادثة الصوت الجماعية المباشرة في ساحات اللعب.
- **قواعد البيانات والأمان (Database & Security)**:
  - JWT (JSON Web Tokens) لتأمين وحفظ الجلسات.
  - bcrypt لتشفير كلمات المرور في مخزن الخادم.
  - محول ذكي يدعم استهلاك قواعد بيانات PostgreSQL (مثل تزويد Neon PostgreSQL) مع وجود آلية تصفح محلية (`db_store.json`) لضمان تشغير فوري 100% دون إعدادات معقدة.
- **الذكاء الاصطناعي**:
  - دمج مع Google Gemini API لتوليد وتلقيم الخادم بأسئلة مميزة ومتماشية تلقائياً.

## 🛠️ إعداد متغيرات البيئة `.env`

قبل النشر المباشر على Render أو تشغيل الحاوية، تلمس تهيئة الملف وتحديث المتغيرات:

```env
# مفتاح خادم الذكاء الاصطناعي من جوجل لتوليد الأسئلة الذكية تلقائياً
GEMINI_API_KEY="أدخل_مفتاح_Gemini_هنا"

# مفتاح التوقيع الخاص بالجلسات المشفرة لعضويات اللاعبين
JWT_SECRET="ClashMindsSuperSecret88"

# بيانات الدخول لمدير اللعبة الافتراضي للتحكم بلوحة المراقبين
ADMIN_EMAIL="admin@siraa.com"
ADMIN_PASSWORD="AdminSiraa2026"

# رابط قاعدة البيانات في حال رغبتك بالربط المباشر مع Neon PostgreSQL
DATABASE_URL="postgresql://..."
```

## 🐳 التشغيل المباشر عبر Docker

لتشغيل المشروع بالكامل بحاوية معزولة بأمر واحد:

```bash
# بناء الحاوية وتشغيلها مباشرة
docker build -t siraa-clash .
docker run -p 3000:3000 siraa-clash
```

## 🎖️ دليل لوحة إدارة اللعبة (Admin Dashboard)

1. توجه إلى الزر المزين برمز **الدروع الذهبية** في أعلى يمين واجهة اللعبة.
2. أدخل الرمز السري للإدارة وهو: `AdminSiraa2525` أو الرمز الافتراضي المعتمد بالخلفية وهو `AdminSiraa2026`.
3. ستاح لك شاشة كاملة لمراقبة:
   - الغرف النشطة وحذفها فوراً.
   - حظر اللاعبين المخربين وفك الحظر عن بريدهم.
   - حقن أسئلة جديدة وتحديث تصنيفاتها.
   - تتبع إحصائيات الاقتصاد من العملات والجواهر.

مكتبة الأسئلة ملقمة تلقائياً بمئات من المسابقات الإسلامية والمناهج الفكرية وعالم الألعاب والمسلسلات. بالتوفيق في المعارك!
